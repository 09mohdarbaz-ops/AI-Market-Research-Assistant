# Market Intelligence Report Builder

A Streamlit app that generates a structured market report from sidebar inputs for industry, country, and forecast period.

## Run

```powershell
pip install -r requirements.txt
streamlit run app.py
```

The app includes sections for Executive Summary, SWOT, PESTLE, Porter's Five Forces, Market Segmentation, and Future Outlook. Reports can be exported as Word or PDF files.
